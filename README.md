## Installing Libraries 
### SDL
`mkdir build`  
`cd build`  
`cmake .. -DCMAKE_BUILD_TYPE=Release`  
`cmake --build . --config Release --parallel`  
`sudo cmake --install . --config Release`  

`Installing: C:/Program Files (x86)/SDL2/..`
